# da_vinciho_kod
Program na zašifrovanie/dešifrovanie textu a vypísanie výsledku do súboru

Pre správne fungovanie programu treba stiahnuť .txt súbor!
Kvôli zle formulovanému a nejasnému zadaniu, nikde nie je špecifikované, ako má program zistiť, či má používateľ záujem o zašifrovanie, alebo dešifrovanie, z toho dôvodu pri vstupe do kolonky "operácia" napísať buď "desifrovanie" alebo "zasifrovanie".
